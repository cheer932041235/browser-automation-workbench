# Examples

This folder contains reusable browser automation examples and task templates.

Examples should be safe, synthetic, and free of private trace data.

## Available Examples

- `record-review-extract.md` — full Browser Intelligence loop
- `engine-page-inspection.md` — direct Browser Engine exploration
- `new-site-profile.md` — checklist for turning repeated browsing patterns into a site profile

## Rule

Do not commit real screenshots, cookies, storage dumps, or trace folders.
