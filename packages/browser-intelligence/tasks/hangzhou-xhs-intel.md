# 内容任务：杭州小红书本地情报

## 任务定位

这是 Browser Intelligence 的内容侧任务之一，不是工具本身。

目标：从小红书等平台中发现对用户现实生活有用的杭州本地机会，尤其是低流量但高价值的信息。

## 关注范围

### 活动

- 杭州周末活动
- 展览/市集/演出
- AI/创业/技术沙龙
- 高校讲座
- 园区开放日
- 读书会/运动局/桌游局/社交局

### 本地资源

- 免费场馆
- 公共空间
- 图书馆/美术馆/博物馆
- 高校公开活动
- 园区资源
- 青年人才活动

### 权益与福利

- 消费券
- 免费体验
- 新店试吃
- 会员权益
- 城市福利
- 政府/园区补贴
- 学生/青年优惠

## 核心价值

用户不喜欢日常刷小红书，但希望不错过：
- 低流量高价值帖子
- 杭州本地活动机会
- 能扩大社交圈的线下活动
- 有实际利益的权益信息
- 能让在杭州生活“不觉得亏”的资源

## 采集策略

### 低频个人助手原则

不做高频爬虫，不做大规模采集。

推荐节奏：
- 每次 3-5 个关键词
- 每个关键词打开 5-10 条候选
- 总计 20-50 条以内
- 由 AI 筛选出 3-10 条真正有价值信息

### 浏览器策略

优先使用真实浏览器登录态：
- Browser Engine / CDP
- 用户手动处理登录或验证
- 遇到验证码不绕过
- 保留截图和来源链接

## 评分标准

| 维度 | 说明 |
|---|---|
| 时间有效性 | 是否近期还能参加 |
| 地点可达性 | 是否在杭州且交通可接受 |
| 社交价值 | 是否有认识人的机会 |
| 资源价值 | 是否有权益、福利、免费资源 |
| 真实性 | 是否像真实用户分享，而非营销号 |
| 稀缺性 | 是否低流量但高价值 |
| 行动成本 | 报名/费用/交通是否麻烦 |

## 输出分级

### A 级：建议行动

适合立刻报名、收藏、加入日历或联系。

### B 级：值得收藏

有价值，但时间/地点/行动成本暂不确定。

### C 级：参考信息

可以拓宽认知，但短期不行动。

### D 级：无效或营销

营销号、过期、信息不完整、价值低。

## 风险边界

不做：
- 高频批量爬取
- 绕过验证码
- 私信骚扰用户
- 采集个人隐私
- 公开分发数据集

只做：
- 个人信息整理
- 低频搜索
- 来源链接保留
- AI 总结与行动建议
