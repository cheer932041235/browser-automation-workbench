# Tasks：内容任务模板

## 定位

本目录存放内容侧任务模板。

任务模板说明“要做什么、怎么判断价值、输出成什么格式”，但不保存具体运行结果。

## 已有模板

| 文件 | 用途 |
|---|---|
| `hangzhou-xhs-intel.md` | 杭州小红书本地活动、权益、社交情报 |
| `generic-site-research.md` | 通用网站调研与自动化复盘 |

## 任务模板应该包含

- 任务目标
- 输入来源
- 搜索关键词或入口
- 成功标准
- 评分标准
- 输出格式
- 风险边界
- 归档建议

## 任务模板不应该包含

- 大量原始采集内容
- 账号 Cookie/Token
- 临时截图
- 每次执行的 trace
- 未清洗的网络请求体

## 输出去向

运行输出：

```text
logs/browser-intelligence/
```

长期笔记：

```text
notes/
```
